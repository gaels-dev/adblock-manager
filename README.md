# AdBlock Manager

A Spring Boot application for managing domain blocklists and generating ad-blocking configurations.

---

## Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* Hibernate
* Maven
* MySQL

---

## Goals

* Manage blocked domains
* Organize domains by categories
* Generate Hosts file configurations
* Generate Pi-hole compatible blocklists
* Explore DNS filtering concepts

---

## Project Status

Currently under development.
